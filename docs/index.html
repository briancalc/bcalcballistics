Bcalc External Ballistics Calculator

open source external ballistics calculator

Audience

  • Hunters and hobbyists? Yep [practical ballistics data for typical field sencarios]
  • Long-range precision shooters? Nope [check out Hornady's excellent 4DOF calculator]

Features

  • enter specific ammo & environmental factors for up to 3 rounds
  • calculator returns output tables for trajectory metrics such as drop, wind, and velocity
  • and draws 3 graphs comparing drop, velocity, and kinetic energy
  • plus creates text files containing all calculated metrics for integer ranges 1 to 800 yards

Installation

(1) via pip: pip install bcalcballistics

(2) download from github
